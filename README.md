## Features
- **Text & Voice Translation**
- **Offline Mode** (50+ languages)
- Real-time camera translation
- Phrasebook with favorites
- Language detection

## Tech Stack
- Android SDK (API 21+)
- Google ML Kit
- Firebase Cloud Translation
- Room Database
